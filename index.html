Projeto em Destaque

Infra Monitoring System

Projeto criado para consolidar fundamentos de automação, métricas e pipelines. Coleta métricas locais com psutil, expõe dados para Prometheus, utiliza Docker para reprodutibilidade e emprega GitHub Actions para testes, análise de segurança e validações.

O foco do projeto é educacional, reforçando práticas reais como organização de código, documentação, testes automatizados e integração de ferramentas essenciais do ecossistema moderno de desenvolvimento.

Processos & Práticas Profissionais

  • Versionamento completo com Git: branches, PRs e histórico limpo.
  • Pipelines no GitHub Actions com testes, lint, análise de segurança e build.
  • Testes automatizados com Pytest e medição de cobertura.
  • Qualidade e segurança: Ruff, Isort, Bandit, Gitleaks, Snyk, Trivy e Dependabot.
  • Containerização com Docker e serviços via Docker Compose.
  • Documentação técnica estruturada em PT e EN.
  • Ambientes isolados e reprodutíveis com venv.
